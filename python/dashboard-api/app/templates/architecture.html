Communication Legend

  • Sync calls: POST /message:send
  • Streaming calls: POST /message:stream + SSE events
  • Discovery: GET /agents/search + GET /.well-known/agent-card.json
  • Data plane: SQLite reads/writes from Python services

Mission Hop Sequence

  1. Command refreshes registry and discovers scout_system.
  2. Intelligence streams status updates and publishes intelligence-report.
  3. Tactical returns deterministic threat and risk scoring.
  4. Logistics computes transport, fuel, and troop movement.
  5. Communications relay secures and encodes the payload.
  6. Fleet streams deployment phases and final deployment-order.
